Sobre o projeto:
~~~~~~~~~~~~~~~~

- Aplicativo feito em React Native (RN)
- Para a execução do código foi utilizado Node.js v14.17.3, e para gerenciar o projeto foi utilizado npm v6.14.13
- Para o desenvolvimento, foi utilizado a plataforma e framework Expo para facilitar baterias de testes em diversos dispositivos
- Para emular na máquina, foi utilizado Android Studio


Instruções sobre execução:
~~~~~~~~~~~~~~~~~~~~~~~~~~

- Para rodar RN utilizou Noje.js: https://nodejs.org/en/
- Para tirar o melhor proveito, na hora de realizar testes de execução, pode-se utilizar Expo: https://docs.expo.io/
- Como foi utilizado RN, é necessário emular o program em um iOS/Android, na minha máquina foi utilizado Android Studio: https://developer.android.com/studio
